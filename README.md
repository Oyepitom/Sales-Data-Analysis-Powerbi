Project Overview. This project analyzes a sales dataset to identify trends, performance patterns, and key insights to support business decision-making.  
Tools Used - Power BI - Data Cleaning - Data Visualization.  
Key Tasks - Cleaned and prepared raw data - Created data model in Power BI - Built interactive dashboard using filters and slicers.  
Key Insights - Identified top-performing products contributing most to revenue - Analyzed sales trends over time - Highlighted performance differences across categories. 
Files Included - Power BI dashboard (.pbix) - Dashboard screenshots.
